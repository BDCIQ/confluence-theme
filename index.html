PharmaSEE Qlik Application User Guide

This guide is designed to help users understand, access, and navigate the PharmaSEE Qlik application. Whether you're new to the platform or looking to deepen your understanding, this page provides step-by-step instructions, tips, and support resources.

PharmaSEE Overview

PharmaSEE is a secure, cloud-based analytics platform developed by PharmalytIQ. It provides interactive dashboards and reports that help users:

  • Monitor pharmacy performance
  • Analyze patient trends
  • Track financial metrics
  • Ensure compliance with 340B and other programs

The application is built on Qlik and is structured into packages based on user roles and data access needs.

How to Access PharmaSEE

To access the PharmaSEE Qlik application:

  1. Go to https://pharmalytiq.com
  2. Click on the “Client Login” button in the top-right corner.
  3. Log in using your corporate credentials (email and password).
  4. Complete MFA (Multi-Factor Authentication) if prompted.

Note: First-time users must set up MFA within 48 hours of receiving access.

How to Request Access

Access is provisioned through the support portal. Please do not email or message individuals directly.

To request access:

  1. Visit https://sackscorp.atlassian.net/servicedesk/customer/portal/2/
  2. Select the “Access Request” form.
  3. Fill in the required fields:
    • Your name and email
    • Your role and department
    • Reason for access
  4. Submit the ticket.

You will receive a confirmation email and further instructions once your request is processed.
